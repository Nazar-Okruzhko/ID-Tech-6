# Wolfenstein-2-Resources

<img width="785" height="568" alt="Screenshot (1222)" src="https://github.com/user-attachments/assets/d66e5424-3615-432c-906e-51d9550b76eb" />


A legendary collections of Python scripts to extract all of the assets from WTNC/WTYB, including 3D models, textures, sounds, scripts and even the declaration files, Assets are stored in proprietary formats like md6mesh, texdb, wem/bnk, decl. Turns out the Decompressing method is Oodle 5. Supports loading for .reasources files .txdb not implemented yet, tested with WTNC, extracted all the resources already.
